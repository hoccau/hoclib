############################################
## hoclib - Bibliotheque PureData vanilla ##
############################################

Ensemble d'objets écrits en puredata. 
Le répertoire pricipal contient les objets hoc.*.pd, sans graph-on-parent.
Le répertoire "views" contient les objets h.*.pd, avec interface graphique. 
Ces deux répértoires sont à ajouter dans le PATH de puredata. 

Dépendances : 
-> puredata vanilla
-> expr, expr~
