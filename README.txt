############################################
## hoclib - Bibliotheque PureData vanilla ##
############################################
Développée selon les besoins persos courants & pour Addictive Atari Sound System.

Tous les objets sont développés en PureData Vanilla. 
Convention pour chacun des objets hoc.* :
-> Sans GUI ni graph-on-parent
-> Les paramètres sont accessibles par arguments & inlets
-> Les paramètres sont bornés dans les valeurs correctes
-> Sans argument, des valeurs sont affectés pour que l'objet fonctionne

Dépendances : 
-> puredata vanilla
-> expr, expr~
